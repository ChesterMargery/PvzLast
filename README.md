# PvzLast

PvzLast 是一个基于 AVZ (AsmVsZombies) 框架的植物大战僵尸游戏终极优化项目。本项目利用 AVZ 项目的资源和技术，为游戏提供高效的脚本编写和自动化操作支持。

## 项目简介

PvzLast 旨在提供一个完整的 PvZ 游戏优化解决方案，包括：

- 高效的游戏内存操作框架
- 便捷的脚本编写接口
- 丰富的工具函数库
- 完整的游戏数据结构定义

## 功能特性

- **内存管理**：提供安全的游戏内存读写接口
- **脚本支持**：支持自定义游戏脚本的编写和执行
- **数据结构**：完整的游戏内部数据结构定义
- **工具函数**：丰富的辅助函数用于简化开发

## 环境要求

- Windows 操作系统
- Visual Studio 2019 或更高版本
- C++17 或更高标准支持
- 植物大战僵尸游戏本体

## 安装说明

1. 克隆本仓库：
   ```bash
   git clone https://github.com/ChesterMargery/PvzLast.git
   ```

2. 使用 Visual Studio 打开项目

3. 配置项目设置：
   - 设置 C++ 语言标准为 C++17
   - 配置包含目录和库目录

4. 编译项目生成 DLL

## 使用方法

1. 启动植物大战僵尸游戏
2. 运行编译生成的工具
3. 按照脚本说明进行操作

## 项目结构

```
PvzLast/
├── src/                    # 源代码目录
│   ├── core/              # 核心功能模块
│   ├── memory/            # 内存操作模块
│   └── scripts/           # 脚本模块
├── include/               # 头文件目录
├── docs/                  # 文档目录
└── examples/              # 示例脚本
```
